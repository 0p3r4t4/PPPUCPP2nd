Para instalar gcc en versiones superiores a las del sistema Ubuntu:

    sudo add-apt-repository ppa:ubuntu-toolchain-r/test
    sudo apt-get update
    sudo apt-get install gcc-6 g++-6
    sudo apt-get install gcc-5 g++-5
-------------------------------

Para compilar con la última versión de g++ en la implementación más reciente del
estándar ISO (sin entrar en detalles de optimización, arquitectura o tal):

    g++-6 -std=c++1z -Wall -o <ejecutable> <código fuente>
-------------------------------

El stream cout permite la salida de variables de tipo bool. Por defecto escribe 1 para
true y 0 para false, pero podemos hacer que escriba true o false con el modificador
std::boolalpha:

    bool b = true;
    cout << b;

    resultado: 1

    bool b = true;
    cout << boolalpha << b;

    resultado: true
-------------------------------

Podemos hacer que el stream cout escriba según un locale concreto o según el locale
del entorno de ejecución del programa, simplemente con:

    cout.imbue(locale(""));
-------------------------------

La "inicialización uniforme y universal" mediante el operador {} impide las conversiones
implícitas de estrechamiento:

    double x {2.7};     // OK
    int y {x};          // ERROR: double -> int night narrow
    char b {1000};      // ERROR: literal value produces narrowing
-------------------------------
